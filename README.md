# zScroll
基于jQuery的自定义滚动组件，支持chrome,edge,safari,fireFox,ie8及以上浏览器

## About

  zScroll.js is a small JavaScript library making dom scroll

## Installation
  
  need npm cmd

    $ npm install zScroll

  With a stand-alone build

    <script src='zScroll.js'></script>


## Example
  
    <script src='zScroll.js'></script>
    $(function(){
      $('#scroll_wrap').zScroll();
    })
